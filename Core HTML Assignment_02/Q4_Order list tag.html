Tag                   Description

  1. h1 to h6 :-It defines headings for an HTML document from level 1 to level 6.
  2. p :- This is the Paragraph tag
  3. hr :- hr stands for Horizontal Rule. This tag is used to put a line across the webpage.
  4. br :- br stands for break line, it breaks the line of the code.
  5. b :- This tag makes our tag bold
  6. i :- i tag makes text italic.
  7. u :- u tag underlines the tag.
  8. em:- em stands for emphasis
  9. big:- big tag makes the text big.
  10. small:- small tag makes our text small.
  11. a:- Anchor tag gives link between web pages.
  12. sup:-Makes our text super scripted.
  13. sub:- Sub script tag sub scripts our Text.
  14. buton:-It is used to represent a clickable button.
  15. body:-It is used to define the body section of an HTML document.
  16. canvas:-It is used to provide a graphics space within a web document.
  17. div:-It defines a division or section within HTML document.
  18. dl:-It is sued to define a description list.
  19. dd:-It is used to provide definition/description of a term in description list.
  20. embed:-It is used as embedded container for external file/application/media, etc.
  21. footer:-It defines the footer section of a webpage.
  22. header:-It defines the header of a section or webpage
  23. img:-It is used to insert an image within an HTML document.
  24. nav:-It represents section of page to represent navigation links.
  25. table:-It is used to present data in tabular form or to create a table within HTML document.
